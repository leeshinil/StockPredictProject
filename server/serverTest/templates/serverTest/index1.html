BigData 기반으로 DeepLearning 학습을 통한 실시간 주가 예측 시스템

<프로젝트 요약>
주식 데이터를 제공하는 대신증권의 공개 API인 CYBOS PLUS를 활용하여 과거 데이터 및 실시간 데이터를 요청해 시계열 데이터 형태로 저장하고, DeepLearning(LSTM모델)을 시켜 미래예측을 위한 모델을 만듭니다. 주식 투자자에게 실시간으로 1분 단위의 주가를 보여주고, 학습된 모델을 통해 1분 뒤의 예측주가를 계속 갱신하여 보여주는 서비스입니다. 또한, 대략적인 주가 변동을 확인하기 위하여 추가로 CandleChart를 자체적으로 보여주는 서비스입니다.

<프로젝트 구현 사항>
- 실시간 주가 예측 : IT 상위 10개 기업의 현재가와 거래량을 CYBOS API를 통해 금년도 초부터 현재까지 1분 단위로 실시간으로 데이터를 수집하는 자동화 Crawling 프로그램을 구축하였고, 수집한 데이터를 가져와 Tensorflow와 Keras 및 여러 Library를 이용하여 딥러닝 기법을 사용하여 1분 뒤의 예측값을 보여주고 이를 Django Server를 통해 실시간 데이터와 예측 데이터를 시각화하는 서비스입니다.
- 데이터 시각화 : 대신증권 API인 CYBOS PLUS를 사용하여 상위 IT기업 10개를 금년도 1월 1일부터 현재까지 주식 종가, 시가, 최고가, 최저가 등의 방대한 양의 데이터를 한눈에 파악할 수 있도록 캔들 차트로 보여주는 서비스입니다.
